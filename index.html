Distributor–Retailer networking

The Distributor–Retailer Networking System is a network simulation project developed using Cisco Packet Tracer to demonstrate reliable communication between distributors and retailers through a structured and secure network infrastructure.

The project focuses on designing and configuring network topology that enables efficient data exchange across multiple locations. The system includes configuration of routers, switches, and end devices, implementing IP addressing, subnetting, and routing protocols to ensure stable connectivity. It simulates how business entities can share information such as inventory updates, order details, and communication data in a controlled network environment.

Interactive Alphabet Learning Toy

The Interactive Alphabet Learning Toy is an educational hardware project developed to help children learn alphabets through visual interaction.

The system allows users to press alphabet keys, and the corresponding letter is displayed by glowing on a custom-built 8×8 LED matrix. The project integrates an ESP32 microcontroller, logic ICs, and a handmade LED matrix to control the illumination of LEDs in specific patterns representing each alphabet. The design demonstrates understanding of digital electronics, microcontroller programming, and circuit integration to create an engaging and interactive learning tool.

Accident Detection & Emergency Alert System

This IOT based system continuously monitors the movement of a vehicle and automatically identifies accident situations based on sudden abnormal changes.

When a crash is detected, the device immediately sends an emergency alert along with the victim’s live location to nearby hospitals, police authorities, and registered family members. This helps emergency services quickly locate the accident site and provide faster medical assistance, which can be critical in saving lives. The system reduces dependency on manual reporting, especially in situations where the victim is unable to call for help.

Spring's SHM Measuring Device

The Spring SHM Measuring Device is a physics-based experimental system designed to measure and observe Simple Harmonic Motion (SHM) in a spring-mass setup.

The device helps visualize oscillatory motion and determine parameters such as time period and frequency through practical measurement. The project demonstrates how theoretical concepts of periodic motion can be applied using a physical system, allowing accurate observation of displacement and oscillation behaviour. By converting experimental readings into measurable outputs, the device helps in better understanding motion characteristics of springs under different loads.
